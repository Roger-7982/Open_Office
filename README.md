# Open_Office
using C++ to control office easily
Use C++ COM component to quickly control Excel and Word, simplify Office secondary development on Windows.

## Environment
- Windows 10/11
- MSVC compiler
- Microsoft Office installed
- CMake build system

## Build
1. Clone this repository
2. Configure CMake in CLion
3. Build and run the demo project
